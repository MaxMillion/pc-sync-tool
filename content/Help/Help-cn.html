新手上路

  • 连接准备

    • 操作系统

      电脑操作系统

      windows(32/64): XP/Vista/Win7/Win8

      Linux(32/64): Ubuntu Fedora

      Mac(32/64): Snow leopard/lion/moutain lion/mavericks

      手机操作系统

      FirefoxOS V1.2或以上版本

      浏览器版本要求

      Firefox26或以上版本

    • 连接方式(二选一)

      USB连接

      1. 用USB线连接电脑和手机

      2. 手机端通过“设置”，打开remote debugger选项

      3. 手机端启动PCsyncclient应用

      WIFI连接

      1. 手机端连接WIFI

      2. 手机端打开PCsyncclient应用，此时界面上显示wifi-code

      3. 打开Firefox，地址栏输入about:ffos后，回车

      4. 选择WIFI连接，输入wifi-code后，点击“连接”

常见问题

  • FirefoxOS 手机和电脑无法通过USB建立连接

    • 检查浏览器版本是否满足要求
    • 检查 ffosassistant 扩展是否正确安装或启用
    • 检查 USB 数据线和 USB 接口是否损坏或接触不良
    • 检查手机驱动是否正确安装
    • 检查手机端应用 PCsyncclient 应用是否已启动
    • 检查手机设置
    • 检查 ADB server 端口是否被其它手机类助手占用

  • 如何为手机下载和安装驱动程序？

  • FirefoxOS 手机无法和电脑通过WIFI建立连接

    • 检查浏览器版本是否满足要求
    • 检查 ffosassistant 扩展是否正确安装或启用
    • 检查手机是否已经连接WIFI？ 手机和电脑是否连接在同一局域网
    • 手机是否可以获取到WIFI连接码？
    • 检查手机设置

  • Firefox 附加组件一直提示需要重新启动

    • 关闭 Firefox
    • 启动任务管理器，查找并结束"USBMonitor.exe" 和 "ffosadb.exe" 进程
    • 启动 Firefox

  • 提示关闭手机中其它应用程序？

    • 长按手机功能键(Home键)，出现正在运行的应用程序列表
    • 逐个向上拖动，关闭所有的应用程序
    • 重新启动 PCsyncclient 应用
