Connection

  • If you have installed some mobile device assistants, such as tencent or 360 mobile assistant etc.. Sometimes the adb server can't be launched. In taskmanager, you can see some processes look like *adb*.exe, such as tadb.exe wadb.exe and so on. Make sure killing all these processes.
  • In settings application on mobile, just disable "Screen Lock".
  • Sometimes system garbage collector will close PCSyncClient application on mobile for recycling memory, which will break down the communication between mobile and PC. So, just relaunch PCSyncClient application on mobile and make sure it's running during syncing.
  • Click HERE download and install ADB device driver.
