Tours新手上路

设备和系统配置

  • 确保已安装了桌面版 Firefox 28+

    安装 Firefox

  • 为桌面版 Firefox 安装附加组件 Firefox OS Assistant

    安装 Firefox OS Assistant

  • 确保你的设备运行 Firefox OS 1.3+
  • 确保你在设备上安装了 PC 连接器应用

WiFi 连接

  • 快速设置
    • 确保你的设备和电脑连接到同一WiFi网络
    • 在你的设备上运行 PC 连接器
    • 运行桌面版 Firefox 并且打开 Firefox OS 手机助手页面
    • 在 Firefox OS 手机助手页面输入 Wi-Fi 连接码，并点击页面上的连接按钮
    • 现在你可已在桌面浏览器中管理你的设备了：

USB 连接

  • 在Windows上，安装设备驱动
    • 请优先选择安装你的手机制造商提供的驱动程序
    • 安装 Driver Helper 附加组件，可以为你安装第三方提供的设备驱动

      安装 Driver Helper

    • 对于很少的一部分设备，需要拷贝 adb_usb.ini 到当前账户的 .android 目录(例如："C:\Users\用户名\.android\")，并重启你的Windows

      下载 adb_usb.ini

  • 为桌面版 Firefox 安装附加组件 ADB Helper

    安装 ADB Helper

  • 配置你的设备
    • 进入设备的“Settings”，停用“Screen Lock（Settings > Screen Lock）”
    • 进入设备的“Settings”，启用“Remote Debugging（Settings > Device information > More information > Developer）”
    • 通过 USB 接口将设备连接到你的电脑
    • 在设备上运行 PC 连接器应用
  • 运行桌面版 Firefox 浏览器，并且打开 Firefox OS 手机助手
    • 现在你应该可以在 Firefox OS 手机助手的底部看到设备的名称，点击它
    • 现在你可已在桌面浏览器中管理你的设备了：
