Quick start

  • Prerequisites
    • Install Firefox browser v28.0 release. (Download)
    • Install addon of Firefox OS Assistant into Firefox browser. (Download)
  • USB connection
    • Install the device driver
      • Windows

        Download device driver, and run the install.exe.

      • Linux

        Enter the following command:

        sodu vim /etc/udev/rules.d/51-android.rules

        Add the following line:

        SUBSYSTEM=="usb",ATTRS{idVendor}=="05c6",MODE=="0666"
      • Mac

        None.

    • Set up your mobile phone
      • Enable “Remote debugging”
      • Disable ”Screen lock”
    • Launch Firefox browser and run the addon Firefox OS Assistant
    • Launch the application PC Sync on the mobile phone
    • Now you’re ready to manage the mobile phone in USB connection model as follow:

      For any problem, follow instructions from FAQs

FAQs

  • How can I check out the version of Firefox browser?
  • How can I find out whether the device driver has been installed properly?
  • How can I remove the running application on the mobile phone?
    • Long press Home button
    • Trigger the list of running applications and swipe upward the hierarchy of every application to remove it
  • How can I remove mobile phone assistant on the PC?
    • Open “Task Manager” by right-clicking the taskbar --> “Start Task Manager”, or press Ctrl+Shift+Delete
    • Click the “Process” tab
    • Click the processes of mobile phone manager or assistant, and click “End Process”
  • Cannot read the pictures, music and videos on the computer in USB connection?
    • Make sure you have disabled “USB storage” on the mobile phone. (Referring to Set up your mobile phone)
  • Why the application PC Sync shut down automatically?
    • Disable “Screen lock”.

      (Referring to Set up your mobile phone)

    • Re-launch the application and reconnect it on the PC. It will work.
  • What can I do when the connection is broken down on PC?
    • Make sure the application PC Sync in running on the mobile phone. If not, please re-launch it. (Follow the instruction of this question)
    • In USB connection, you can follow the steps below to recover the connection:
      • Remove the processes of other mobile phone assistant running on the PC. (Follow the instruction of this question)
      • As the running assistants occupy the USB server who handles the device-computer connection and communication, they will cause disconnection between your mobile phone and PC.
  • Where is the addon icon of Firefox OS Assistant when I finish install?
    • Make sure the addon Firefox OS Assistant has been install properly
    • Pin the addon icon to the tool bar from the customize-box
