ToursGet started

Device and system configuration

  • Make sure you have Firefox Desktop 28+ installed. (Download)
  • Install the Firefox OS Assistant add-on in Firefox Desktop.
  • Make sure your device is running Firefox OS 1.3+
  • Make sure you have PC Sync installed on your device

WiFi connection

  • Quick setup
    • Make sure your device is connected to Wi-Fi network
    • Launch the PC Sync on your device
    • Launch Firefox browser and run the Firefox OS Assistant
    • Input the Wi-Fi connection code on the Firefox OS Assistant, Click connection button on it.

USB connection

  • Install the device driver
    • Windows drivers (Download)
    • Linux, add vendorId to system, for example:

      Enter the following command:

      sodu vim /etc/udev/rules.d/51-android.rules

      Add the following line:

      SUBSYSTEM=="usb",ATTRS{idVendor}=="05c6",MODE=="0666"
    • Mac

      None.

  • Install the ADB Helper add-on in Firefox Desktop.
  • Set up your device
    • In the Settings of your device, disable Screen Lock (Settings > Phone lock > Lock Screen)
    • In the Settings of your device, enable Remote Debugging (Settings > Device information > More information > Developer)
    • Connect your device to your machine via a USB cable
    • Launch the PC Sync on your device
  • Launch Firefox browser and run the addon Firefox OS Assistant
    • You should see the name of your device in the Firefox OS Assistant's bottom bar. Click on it.
    • Now you’re ready to manage your device as follow:
