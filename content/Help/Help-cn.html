快速启动

  • 环境准备
    • 安装 Firefox 浏览器26.0正式版 （点击下载）
    • 安装 FirefoxOS 手机助手附加组件（点击下载）
  • USB连接
    • 安装驱动
      • Windows

        下载驱动并安装

      • Linux

        运行 sudo vim /etc/udev/rules.d/51-android.rules

        添加新行：

        SUBSYSTEM=="usb",ATTRS{idVendor}=="05c6",MODE=="0666"

      • Mac

        无需配置

    • 设置手机
      • 打开“远程调试”
      • 停用 USB 存储模式
    • 启动 Firefox, 并运行手机助手
    • 启动手机端PC同步应用
    • 用 USB 线连接手机和电脑，连接成功如下图所示，否则见常见问题
  • WIFI连接
    • 修改手机设置
      • 停用“锁屏”功能
      • 停用“休眠”功能
    • 确认手机连接到 WIFI 网络，获取连接码
    • 启动 Firefox, 并运行手机助手
    • 选择 WIFI 连接模式，输入手机端 WIFI 连接码
    • 点击“连接”按钮，连接成功如下图所示，否则见常见问题

常见问题

  • 如何检查 Firefox 浏览器版本?
  • 如何检查手机驱动是否正确安装？
  • 如何关闭手机端正在运行的应用？
  • 如何关闭手机端正在运行的其他手机助手？
  • 为何输入正确的 WIFI 连接码后，无法建立 WIFI 连接？
    • 确认手机和PC运行在同一网络
    • 确认手机端PC同步应用正在运行
  • 为何USB连接时，无法显示手机上的图片、音乐和视频？
    • 检查是否打开远程调试,参考（设置手机）
    • 检查是否关闭了USB存储模式,参考（设置手机）
  • 手机端应用自动退出怎么办？
    • 参考WIFI模式下 设置手机
    • 重启手机端应用
  • PC 端操作时，连接突然中断，如何恢复连接？
    • 检查手机端应用是否被关闭。（参考链接）
    • 如果您使用WIFI连接模式，检查网络是否正常、检查手机设置（参考链接）
    • 如果您使用USB连接模式，尝试以下操作:
      • 关闭电脑上其他的手机助手（参考链接）
      • 参考设置手机
  • 安装完手机助手，在浏览器上找不到对应的图标怎么办？
    • 检查附加组件是否已经正确安装
    • 从浏览器的“个性定制”表中将图标放到工具栏上
